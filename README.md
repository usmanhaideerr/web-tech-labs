##  Lab Tasks Overview

### Lab 1: Basic HTML Structure & Formatting
- **Objective:** Learn fundamental HTML5 tags and text styling.
- **Key Tags Used:** `<h1>`-`<h3>`, `<p>`, `<b>`, `<i>`, `<u>`, `<sub>`, and `<sup>`.
- **File:** `Lab1.html`

### Lab 2: Deployment & Hyperlinks
- **Objective:** Create a homepage hub, link multiple files, and deploy using GitHub Pages.
- **Key Features:** Hyperlinks (`<a>`), Image embedding (`<img>`), and GitHub hosting.
- **File:** `index.html` (formerly Lab2.html)
